# sentiment-analysis-project
This is a BERT-based sentiment analysis project.

## Pretrained Model
The trained model is hosted on Hugging Face: [Link to Model](https://huggingface.co/Pranjalii/sentiment-analysis-bert)

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run the Streamlit app: streamlit run app.py

## Notes
- The large model files are hosted on Hugging Face, not included in this repo.
- Ensure you have an internet connection to download the model when running the app.
